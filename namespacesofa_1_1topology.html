Get the triangle index from one of the point of a triangle in a list and a direction.

Remarks
not really generic
Template Parameters
Realscalar type being used for computations
Coordinatestype representing the coordinates in space (in 3D)
VectorCoordinatestype for the container of coordinates
Parameters
trianglePositionspositions of all the points of the triangles set
allTrianglesinformations about the triangle in the topology (a vector of 3 indices, i.e a triangle)
shellTrianglestriangles around the given point (vector of triangles indices)
pointIDindex of the point (from shellTriangles) from which the algo will try to compute the direction
directiondirection in which the algo will find a triangle from the pointID
Returns
index of the triangle in the given direction, if parameters allow it; sofa::InvalidID
