Detailed Description

template<class Deriv, class MassType>
struct sofa::component::mass::AddMToMatrixFunctor< Deriv, MassType >

Helper struct to add entries in a BaseMatrix, based on the type of Mass (MassType).

This class is specialized for Rigid types.

The default implementation assumes it deals with Vec types: Deriv is a Vec type, and MassType is a floating point.
