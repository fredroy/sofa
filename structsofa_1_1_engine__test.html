Detailed Description

Test suite for engine using TestEngine class. This class has a counter which shows how many times the update method is called. Add inputs to engine. Check that the output is updated only if necessary. For this test 3 engines are used.The output of engine1 is linked to the input of engine2 and also the input of engine3. engine2 / engine1 \ engine3
