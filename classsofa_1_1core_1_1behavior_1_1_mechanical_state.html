Detailed Description

template<class TDataTypes>
class sofa::core::behavior::MechanicalState< TDataTypes >

Component storing all state vectors of a simulated body (position, velocity, etc), using the datatype specified in the templace.

The given DataTypes class should define the following internal types:

  • Real
    : scalar values (float or double).
  • Coord
    : position values.
  • Deriv
    : derivative values (velocity, forces, displacements).
  • VecReal
    : container of scalar values with the same API as sofa::type::vector.
  • VecCoord
    : container of Coord values with the same API as sofa::type::vector.
  • VecDeriv
    : container of Deriv values with the same API as sofa::type::vector.
  • MatrixDeriv
    : vector of constraints.

Other vectors can be allocated to store other temporary values. Vectors can be assigned efficiently by just swapping pointers.

In addition to state vectors, the current constraint system matrix is also stored, containing the coefficient of each constraint defined over the DOFs in this body.
