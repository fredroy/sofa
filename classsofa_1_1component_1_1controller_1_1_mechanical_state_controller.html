Detailed Description

template<class DataTypes>
class sofa::component::controller::MechanicalStateController< DataTypes >

MechanicalStateController Class.

Provides a Mouse & Keyboard user control on a Mechanical State. On a Rigid Particle, relative and absolute control is available.
