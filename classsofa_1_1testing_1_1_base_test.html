◆ BaseTest()

BaseTest ( )

Initialize Sofa and the random number generator.

if you want to generate the same sequence of pseudo-random numbers than a specific test suites use the same seed (the seed value is indicated at the 2nd line of test results) and pass the seed in command argument line ex: SofaTest_test.exe seed 32

gtest already use color so we remove the color from the sofa message to make the distinction clean and avoid ambiguity.

Repeating this for each class is harmless because addHandler test if the handler is already installed and if so it don't install it again.
