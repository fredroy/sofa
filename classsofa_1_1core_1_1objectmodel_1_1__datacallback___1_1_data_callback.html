Detailed Description

Associate to a set of data a set of callback

The callbacks are called when one of the input is changed.

Example of use: Data<int> a; Data<int> b; DataCallback cb;

In constructor: cb.addCallback([&a,&b](DataCallback*){ std::cout << "sum is:" << a.getValue()+b.getValue() << std::endl; });

cb.addInputs({&a,&b}); or cb.addInput(&a); cb.addInput(&b);

a.setValue(5); /// should print: "sum is 5" b.setValue(6); /// should print: "sum is 11"
